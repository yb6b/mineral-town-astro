# 采矿场

☆ 所需道具

- 锄头……挖掘地面。
- 锤子……破坏石头。

☆ 挖掘地面后能够发现的东西

金币  
黑草  
力量之树果实  
陷阱  
前往下层的梯子等

☆ 破坏石头后能够发现的东西

废矿石  
铜  
银  
金  
紫水晶  
钻石  
山铜  
精金等

☆ 能发现的物品效果

- 金币  
  所持金额 UP。
- 黑草  
  食用后回复体力、疲劳度。
- 力量果实  
  食用后最大体力 UP。  
  ※找到就立即食用吧！
- 废矿石  
  没什么特别的。
- 铜、银、金、秘银  
  改造道具所需品。
- 山铜  
  制作饰品所需品。
